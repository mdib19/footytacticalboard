# footytacticalboard
A simple Football Tactical Board. <br>
(This is a personal project built with HTML, CSS and JavaScript, with the assistance of Claude AI.)

## Features:
- Add and drag players freely <br>
- Undo actions <br>
- Clear the board <br>
- Save and load files (.json) <br>


 ## About
This project combines my passion for IT and football. It also serves as a way to strengthen and showcase my frontend development skills as well as familiarity with AI tools and LLMs in development, and applying prompt engineering techniques to improve the output quality.
